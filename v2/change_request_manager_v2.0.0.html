How AI works in this app
AI calls are routed through a Cloudflare Worker proxy that holds your Anthropic API key securely — the key never appears in this HTML file or the GitHub repo.
Setup steps:

1. Go to dash.cloudflare.com → Workers & Pages → Create Worker
2. Paste the contents of worker.js from the repo
3. Settings → Variables → Add: ANTHROPIC_API_KEY → Encrypt → Save
4. Deploy — copy your Worker URL
5. Open this HTML in a text editor, find PROXY_URL and replace YOUR_WORKER_URL_HERE
No API key yet? Get one free at console.anthropic.com → API Keys → Create Key.
